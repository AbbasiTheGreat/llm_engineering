# 📧 AI Email Subject Suggestion

An AI-powered email subject generator that automatically analyzes the content of an email and generates a concise, professional, and context-aware subject line using Google's **Gemini 2.5 Flash** model through the OpenAI-compatible API.

## ✨ Features

- 🤖 AI-generated email subject lines
- 📝 Context-aware subject generation
- 🎯 Concise and professional output
- 🚫 No hallucinated information
- ⚡ Powered by Gemini 2.5 Flash
- 🔄 Simple Python implementation
