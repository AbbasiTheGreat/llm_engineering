# LLM Engineering — Course Notes & Projects

This repository is my personal learning log and project workspace for the
[**LLM Engineering: Master AI and Large Language Models**](https://www.udemy.com/course/llm-engineering-master-ai-and-large-language-models/?couponCode=25BBPMXINACTIVE)
course on Udemy.

I'm using this space to build, break, and rebuild real projects as I go — from
simple API calls to autonomous agentic systems — rather than just following
along passively. Every folder here represents something I've actually run,
debugged, and understood, not just copy-pasted.

## 🎯 Why This Repo Exists

- **Track progress** — a running record of what I've learned week by week.
- **Practice by building** — every concept is paired with a working project.
- **Own the code** — course examples are extended, refactored, or merged with
  my own ideas rather than left as-is.
- **Build a portfolio** — a public trail of applied LLM engineering skills:
  prompting, API integration, agentic workflows, and real business use cases.

## 🧠 What This Course Covers

The course takes you from foundational LLM API usage to building full
autonomous AI agents, including:

- Working with frontier model APIs (OpenAI, and others)
- Prompt engineering and structured outputs (JSON mode, one-shot prompting)
- Multi-step LLM pipelines (chaining calls to accomplish a larger task)
- Retrieval-augmented generation (RAG)
- Fine-tuning open-source models
- Agentic AI design patterns — tools, memory, planning, autonomy
- Real-world business applications for each technique
